# Mini Ecommerce PHP

Projeto de ecommerce desenvolvido com PHP, MySQL, Docker e Nginx.

## Tecnologias

- PHP
- MySQL
- Docker
- Nginx
- HTML/CSS
- JavaScript

## Funcionalidades

- Cadastro de produtos
- Carrinho
- Painel admin
- Upload de imagens
- Sistema de pedidos

## Como executar

```bash
docker compose up -d
```

Acesse:

http://localhost:8080
